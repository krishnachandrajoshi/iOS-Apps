This app is still under developement. It has been implemented as a part of course. App is being further developed to implement more features and make it bug free.
Check the App Screenshots folder for the images of the application.
Features: 
1. Username and password for each individual user.
2. Tags for each task. 
3. Activities can be added under each tag.
4. Date and Time to as reminder

Features under Development:
1. Database for the application.
2. Collaboration with other team members.
3. 'Notes' to add notes under each activity.

NOTE:
User Name and Password has to be created for login and use of application, exceptions for wrong username and password are not taken care of. Cannot login otherwise. 
Please select new User button to create a new user. 
User name and password are case sensitive.
